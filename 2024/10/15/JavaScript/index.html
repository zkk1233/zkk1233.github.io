JavaScript 数组

JavaScript 不支持使用名字来索引数组，只允许使用数字索引

如果你使用名字作为索引，当访问数组时，JavaScript 会把数组重新定义为标准对象。

执行这样操作后，数组的方法及属性将不能再使用，否则会产生错误（person.length输出为0）
